# MediFlow OPD - Complete Package

## Files Included
- server.js - Updated backend with pharmacy routes
- routes/pharmacy.js - Pharmacy API endpoints
- pharmacy_schema.sql - Database tables and sample data

## Installation Steps
1. Extract this ZIP into your project folder
2. Run the SQL schema in your database
3. Upload all files to GitHub
4. Redeploy on Render

## Pharmacy Features
- Drug catalog with 10 sample drugs
- Batch tracking with expiry dates
- FIFO dispensing
- Stock alerts (low stock, expiry)
- Prescription fulfillment
- Reports (stock levels, expiry, dispensing)
